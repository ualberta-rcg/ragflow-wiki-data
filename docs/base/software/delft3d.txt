[[Category:Software]]

[https://oss.deltares.nl/web/delft3d/home Delft3D] is a 3D modeling suite to investigate hydrodynamics, sediment transport and morphology and water quality for fluvial, estuarine and coastal environments.

= Examples =
Delft3D comes with a number of <code>run_*</code> scripts that are expected to be used with the Sun Grid Engine job scheduler and the MPICH library. The Alliance uses SLURM for a [[Running jobs|job scheduler]] and Open MPI for a default MPI implementation. To illustrate how one can run Delft3D under SLURM, we have provided submission scripts to run the examples supplied with the software.

To copy examples into a scratch directory follow these steps:

<pre>
$ module load StdEnv/2020  intel/2020.1.217  openmpi/4.0.3 delft3d
$ cp -a $EBROOTDELFT3D/examples $SCRATCH/delft3d-examples
</pre>

A test case within each sub-directory contains a <code>start-slurm.sh</code> script that you can run like this:
 $ sbatch start-slurm.sh

The <code>readme.examples</code> file provides a summary of the results.
