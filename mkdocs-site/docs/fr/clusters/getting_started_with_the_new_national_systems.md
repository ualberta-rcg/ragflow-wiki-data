Nous vous suggérons de lire d'abord la page [Soutien technique](https://docs.alliancecan.ca/wiki/Technical_support/fr) et de prendre connaissance des webinaires sur comment travailler avec les grappes nationales [Cedar](cedar.md) et [Graham](graham.md).

Les webinaires vidéo, d'une durée de 5 à 10 minutes chacun, se trouvent sur

* [notre site YouTube](https://www.youtube.com/playlist?list=PLlIsA3IgWjnweFiHlOjBmsF2NTeJ6f5rT).

D'autres présentations sont aussi disponibles sur les sujets suivants :

* [Getting an account](https://www.youtube.com/watch?v=EtFUvBYaaZE) : comment obtenir un compte de l'Alliance
* [General info](https://www.youtube.com/watch?v=9Judzg9RbFw) : renseignements généraux sur les nouveaux systèmes
* [Software environment](https://www.youtube.com/watch?v=BteD0aY5zxg) : description de l'environnement logiciel
* [File systems](https://www.youtube.com/watch?v=66_WGFpjwew) : description des systèmes de fichiers
* [Managing jobs](https://www.youtube.com/watch?v=dfw027j284Q) : gestion des tâches et commandes Slurm de base
* [Common mistakes to avoid](https://www.youtube.com/watch?v=TOYsh0TPy2U) : erreurs à éviter
* [Accessing the environment](https://www.youtube.com/watch?v=VONgiHWc-gE) : accès à l'environnement, de la connexion au transfert de données

Si vous avez besoin d'aide, consultez [la page Soutien technique](../support/technical_support.md).
