*Page enfant de [Service infonuagique de Calcul Canada](cloud-fr.md)*

Les instances (machines virtuelles ou VMs) utilisées pour fournir des données génériques ou du contenu Web sont configurées selon les gabarits de type persistant (voir [Gabarits d'instances](virtual-machine-flavors-fr.md)) et sont [démarrées depuis un volume](working_with_volumes-fr#démarrer_depuis_un_volume.md).

*Pour créer un serveur de données partagées, consultez [Serveurs FTP dans notre environnement infonuagique](ftp-server-in-the-cloud-fr.md) où il est aussi question de HTTP et SFTP.
*Pour créer un serveur web, consultez [Création d'un serveur web dans notre environnement infonuagique](creating-a-webserver-on-the-cloud-fr.md).
