!!! warning "Helios Closure"
    Helios is now closed.

Helios was a supercomputer comprised of Graphics Processing Unit (GPGPU) nodes that was installed in the computing centre at Université Laval in Spring 2014. In anticipation of its decommissioning, interactive access was discontinued on October 26, 2022. Data can be extracted using the Globus endpoint.
