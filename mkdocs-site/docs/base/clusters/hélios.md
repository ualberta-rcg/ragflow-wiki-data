!!! warning "Helios Closure"
    Helios is now closed.

Hélios was a supercomputer composed of Graphics Processing Unit (GPGPU) nodes that was installed in the computing centre at Université Laval in spring 2014. In preparation for decommissioning, interactive access was discontinued on October 26, 2022. Data can be extracted using the Globus endpoint above.
