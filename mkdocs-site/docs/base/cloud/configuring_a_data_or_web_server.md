Virtual machines used to provide generic data or web content typically are of a persistent flavor (see [Virtual machine flavors](virtual-machine-flavors.md)) and [boot from a volume](working_with_volumes#booting_from_a_volume.md).

*If you are interested in creating a server to share data see [FTP server in the Cloud](ftp-server-in-the-cloud.md) which also discusses HTTP and SFTP.
*If you are interested in creating a web server see [Creating a web server on a cloud](creating-a-web-server-on-a-cloud.md).
