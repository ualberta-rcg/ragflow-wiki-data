Each year, the Alliance publishes a report that includes the list of successful RAC projects with a summary of the resources awarded, as well as some details and statistics showing the state of affairs for that particular competition (e.g., overall capacity, resources allocated per cluster and resource, etc.).

The list of projects awarded for each round is included as a link at the top of the corresponding report (the list will open in a separate Google sheet).
