Virtual machines used to provide generic data or web content typically are of a persistent flavor (see [Virtual machine flavors](virtual_machine_flavors.md)) and [boot from a volume](working_with_volumes.md#booting-from-a-volume).

!!! seealso

    *   If you are interested in creating a server to share data, see [FTP server in the Cloud](ftp_server_in_the_cloud.md), which also discusses HTTP and SFTP.
