Virtual machines used to provide generic data or web content typically are of a persistent flavour (see [Virtual machine Flavours](virtual-machine-flavours.md)) and [boot from a volume](working-with-volumes.md#booting-from-a-volume).

* If you are interested in creating a server to share data, see [FTP server in the Cloud](ftp-server-in-the-cloud.md) which also discusses HTTP and SFTP.
